# 🎫 AI-Based IT Ticket Routing & Classification System

## 📌 Project Overview
This project is an AI-powered IT ticket support system that automatically classifies incoming support tickets into:
- 🏢 Department
- ⚡ Priority
- 🏷️ Tags

It helps organizations streamline ticket handling, reduce manual effort, and improve response time.

---

## 🚀 Features
- Automatic ticket classification using Machine Learning
- Multi-label prediction (Department, Priority, Tags)
- Real-time prediction using Streamlit UI
- Text preprocessing using TF-IDF
- Scalable and efficient model

---

## 🧠 Technologies Used
- Python 🐍
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression (Multiclass Classification)
- Jupyter Notebook (Model Building)
- Streamlit (Frontend UI)
- PyCharm (Development Environment)
